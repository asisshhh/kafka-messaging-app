🎯 API ENDPOINTS

POST /api/send
  Send a message to Kafka
  Request:  { "content": "Hello!", "sender": "Alice" }
  Response: { "success": true, "message": "...", "sentAt": "..." }

GET /api/messages
  Get all received messages
  Response: { "count": 3, "messages": [...], "timestamp": "..." }

POST /api/clear
  Clear all messages
  Response: { "success": true, "message": "Cleared X messages" }

GET /api/health
  Server health check
  Response: { "status": "healthy", "messageCount": 3, ... }

🚨 TROUBLESHOOTING

Problem: "Cannot connect to localhost:9092"
Solution: Ensure Docker containers are running
  docker-compose up -d
  docker ps

Problem: "Port 3000 already in use"
Solution: Use different port
  PORT=3001 npm start

Problem: "npm install fails"
Solution: Clear cache and retry
  npm cache clean --force
  npm install

Problem: "WebSocket connection failed"
Solution: Refresh browser, check server is running
